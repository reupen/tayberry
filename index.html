Motivation

It's simple: to have a canvas-based charting library that's really easy to use, configurable and free.

Download

The latest version is 0.2.2, available from GitHub.

Documentation

Head over to github.com/reupen/tayberry for the docs.

Examples

Two histograms overlaid using partial transparency

Of course, you have your histograms one per chart if you prefer.

Show code 
Tayberry.create('chart1', {
    title: {
        text: 'Forecast errors'
    },
    yAxis: {
        title: {
            text: 'Frequency'
        }
    },
    xAxis: {
        title: {
            text: 'Error'
        },
        type: 'linear',
        min: -50,
        max: 50,
        tickStepValue: 10
    },
    presets: ['histogram'],
    series: [{
        data: [11, 13, 20, 19, 37, 38, 43, 49, 53, 80, 84, 102, 122, 111, 115, 143, 161, 148, 196, 210, 213, 220, 226, 227, 235, 189, 233, 201, 188, 158, 177, 145, 120, 101, 112, 104, 69, 71, 52, 38, 31, 23, 17, 13, 12, 12, 9, 6, 6, 4],

Stacked bar chart with negative values

Also using shared tooltips, which display values for all series rather than just the one under the pointer.

Show code 
Tayberry.create('chart2', {
    title: {
        text: 'Contribution to revenue growth'
    },
    xAxis: {
        categories: ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul', 'Aug', 'Sept', 'Oct', 'Nov', 'Dec'],
        title: {
            text: 'Month'
        }
    },
    yAxis: {
        title: {
            text: 'Impact'
        },
        labelFormat: 'currency'
    },
    barMode: 'stacked',
    tooltips: {
        shared: true
    },
    series: [{
        data: [10, 9, 8, 7, 6, 5, 4, 3, 3, 0, -3, -5],
        name: 'Frequency'
    }, {
        data: [-5, -6, -8, -10, -9, -7, -7, -3, -2, 2, 1, 3],
        name: 'Price'
    }, {
        data: [2, 1, 3, -1, -3, -3, -4, 3, 4, 7, 3, 2],
        name: 'Special offers'
    }, {
        data: [5, 6, 7, 6, 2, 2, 10, 6, 1, 2, -3, -6],
        name: 'Transaction size'
    }]
});

Standard bar chart with multiple series

Using data labels and percentage formatting.

Show code 
Tayberry.create('chart3', {
    title: {
        text: 'UK household sizes'
    },
    yAxis: {
        labelFormat: 'percentage'
    },
    xAxis: {
        categories: ['1', '2', '3', '4', '5+']
    },
    labels: {
        enabled: true
    },
    series: [{
        data: [.132, .290, .224, .180, .173],
        name: '1961'
    }, {
        data: [.306, .341, .156, .129, .069],
        name: '2011'
    }]

});

Horizontal bar chart

Using a custom label formatter.

Show code 
Tayberry.create('chart4', {
    title: {
        text: 'UK population by region (mid-2013)'
    },
    xAxis: {
        categories: ['North East',
            'North West',
            'Yorkshire and The Humber',
            'East Midlands',
            'West Midlands',
            'East',
            'London',
            'South East',
            'South West',
        ]
    },
    swapAxes: true,
    yAxis: {
        labelFormatter: function(val) {
            return (val/1000000).toFixed(1) + 'm';
        }
    },
    series: [{
        data: [2610481,
            7103261,
            5337711,
            4598431,
            5674712,
            5954316,
            8416543,
            8792766,
            5377596],
        name: 'Population (people)'
    }]
});
