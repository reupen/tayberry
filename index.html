Mixed positive and negative value stacked bar chart, shared tooltip 
Tayberry.create('chart2', {
    title: {
        text: 'Contribution to revenue growth'
    },
    xAxis: {
        categories: ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul'],
        title: 'Error'
    },
    yAxis: {
        title: 'Impact',
        labelFormat: 'currency'
    },
    barMode: 'stacked',
    tooltips: {
        shared: true
    },
    series: [{
        data: [10, 9, 8, 7, 6, 5, 4],
        name: 'Frequency'
    }, {
        data: [-5, -6, -8, -10, -9, -7, -7],
        name: 'Price'
    }, {
        data: [2, 1, 3, -1, -3, -3, -4],
        name: 'Promotions'
    }, {
        data: [5, 6, 7, 6, 2, 2, 10],
        name: 'Transaction size'
    }]
});
Side-by-side series, with labels

        labelFormat: 'percentage'
    },
    xAxis: {
        categories: ['1', '2', '3', '4', '5+']

        enabled: true
    },
